- #### oque é DOCTYPE
doctype significa "Document Type Declaration" ou em portugues (Declaração do Tipe de Documento). o nome é alto explicativo, o DOCTYPE é a primeira linha de um arquivo [[HTML]] que diz ao navegador: "ei oque vem a seguir é HTML5"

- #### Por que existe?
Nos tempos antigos da web, existiam varias versões do [[HTML]] (HYTML4, XHTML, etc), eo navegador precisava saber qual versão voce estava usando para renderizar a página corretamente. o DOCTYPE era a forma de comunicar isso.

- #### Como é o HTML5 (atual)
`<!DOCTYPE html>`
É literalmente isso, Super simples. Vai sempre ser a primeira linha do arquivo.

- #### Comparação com versões antigas
preste atenção como era uma dor de cabeça antes:

HTML4
```
<!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">
```

XHTML
```
<!DOCTYPE html PUBLIC "-//W3C//DTD XHTML 1.0 Strict//EN" "http://www.w3.org/TR/xhtml1/DTD/xhtml1-strict.dtd">
```

HTML5 (atual)
```
<!DOCTYPE html>
```
