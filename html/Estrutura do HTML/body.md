#### O que é body?
o `<body>` é uma seção do [[HTML]] onde vai ==todo o conteúdo visivel da pagina==.
Enquanto o `<head>` é o "bastidor", o `<body>` é o "palco" tudo que o visitante ve na tela vem daqui.

- #### Estrutura básica
```
<!DOCTYPE html>
<html>
  <head>
    <title>Minha Página</title>
  </head>
  <body>
    <!-- TUDO que aparece na tela vai aqui -->
  </body>
</html>
```

- #### O que pode ir dentro do body ?
Praticamente qualquer elemento visivel:

#### 1.  Títulos ([[<h1> a <h6>]])
```
<h1>Título Principal</h1>
<h2>Subtítulo</h2>
<h3>Seção menor</h3>
```

#### 2. Parágrafos ([[<p>]])
`<p> Este é um parágrafo de texto normal.</p>`

#### 3. Imagens ([[<img>]])
`<img src="foto.jpg" alt="Descrição da imagem">`
